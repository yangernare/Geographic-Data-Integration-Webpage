# Geographic-Data-Integration-Webpage
Browser-based GIS tool aggregating 51 global data platforms (32 Chinese + 19 int'l) covering imagery, DEM, climate &amp; vector data. Features GDAL 3.8.4 WASM for client-side preprocessing, WebLLM local AI assistant. Built with WebGPU, geotiff.js, proj4js. Seamless one-stop workflow for researchers.
